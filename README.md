# Фронтенд для API

## Настройка окружения

### Переменные окружения

Проект использует следующие переменные окружения:

| Переменная | Описание | Значение по умолчанию |
|------------|----------|----------------------|
| `VITE_API_URL` | Базовый URL API | `http://localhost` |
| `VITE_API_TIMEOUT` | Таймаут запросов в миллисекундах | `30000` |

### Файлы окружения

1. **`.env.example`** — шаблон с комментариями (закоммичен в репозиторий)
2. **`.env.development`** — настройки для локальной разработки
3. **`.env.production`** — настройки для продакшена

### Быстрый старт

```bash
# Скопировать шаблон
cp .env.example .env.development

# Отредактировать при необходимости
# Запустить проект
npm run dev
```

### Сборка для продакшена

```bash
# Убедиться, что .env.production настроен корректно
npm run build
```
