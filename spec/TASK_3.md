
### Переменные
- VITE_API_URL — базовый URL API
- VITE_API_TIMEOUT — таймаут запросов (мс)

### Значения по умолчанию
- .env.development: VITE_API_URL=http://localhost
- .env.production: VITE_API_URL=https://your-domain.com
