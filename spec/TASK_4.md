# Текущая задача: Исправить заполнение формы редактирования и добавить просмотр стратегии

## Метаданные
- **Ветка**: fix/strategy-edit-view
- **Проект**: frontend (React + Ant Design + Redux Toolkit)

## Проблемы
1. При нажатии кнопки "Редактировать" в таблице стратегий форма открывается, но поля пустые, хотя запрос на получение стратегии успешно выполняется и данные приходят
2. Нет возможности просмотреть стратегию в режиме "только чтение"

## Требования

### 1. Исправить форму редактирования
- При нажатии кнопки "Редактировать" в таблице:
  - Переход на роут `/strategies/edit/:id`
  - Загрузка стратегии через `fetchStrategy(id)`
  - Форма должна заполниться полученными данными
- Поля формы:
  - description (текстовое поле)
  - parameters (JSON-редактор, TextArea)
  - остальные поля, которые уже заполняются

### 2. Добавить просмотр стратегии
- В таблице стратегий поле `description` сделать кликабельным (ссылка)
- При клике на description открывается страница `/strategies/view/:id`
- Страница просмотра должна содержать:
  - Заголовок "Просмотр стратегии"
  - Все поля стратегии в режиме "только чтение" (disabled или обычный текст)
  - Кнопка "Назад" для возврата в список
- Все данные загружаются через `fetchStrategy(id)`
